# CounterRANT
A mix of Counter Strike and VALORANT (just the core sense of those two games)

---

For the graphics and basic function of the game, I will use Unreal Engine 4, simply because an [Waka/Hackatime Extension](https://github.com/ZXMushroom63/WakatimeIntegration) exists for it and I don't want to fry my PC to death (Ryzen 5300U with integrated graphics...).

I also need to decide if using C++ or Blueprints is better for the logic of the game but it'll stick with Blueprints as it is way simpler than C++. If I notice a lot of things I want to do won't work because I'm not using C++, I'll consider switching (I can't code in C++ so that will probably boost the AI usage).
